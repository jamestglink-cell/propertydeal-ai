Privacy Policy

Effective Date: June 2026

Property Deal AI stores property analysis data locally on the user’s device. The application does not collect, transmit, or share personal information, financial information, or property analysis data with the developer or third parties.

Property Deal AI does not require user accounts and does not use cloud storage in its current version.

If future versions introduce optional cloud storage, AI-powered features, analytics, or other online services, this Privacy Policy will be updated accordingly. Users will be notified of any material changes through app updates or an updated Privacy Policy.

If you have questions regarding this Privacy Policy, please contact the developer through the information provided on the App Store listing.
